# 190826 PPT

## 글꼴
- 고딕체 (맑은고딕, 나눔고딕, 나눔스퀘어 등)
- 제목 크기 : 최소 20pt
- 내용 크기 : 최소 18pt



## 옵션
- 자동 복구 저장 정보 시간 : 10분
- 글꼴 포함 저장



## PPT 제작 스킬

### 그림 편집
- 그림 자르기 : 더블클릭하고 위쪽 메뉴 '자르기'
- 그림 배경 없애기 : 더블클릭하고 위쪽 메뉴에 '배경제거'

### 동영상 편집
- 자동재생 : 화면 넘어가면 재생버튼 클릭하지 않아도 자동재생
- 전체화면 재생 : 재생시키면 자동으로 전체화면 실행
- 하이퍼링크 : 컴퓨터 내부에 있는 파일도 링크해서 실행 가능

### 다운받은 템플릿에 있는 그림 요소 없애기
- 보기 - 슬라이드마스터 - 요소 삭제



## 발표 시 주의사항

### 태도
- 말인사 -> 허리인사
- 질문 받으면 > 이해한 내용이 맞는지 확인 > 답변 > 답변이 되었나 확인
