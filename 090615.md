## 함수?

> 프로그램 명령어의 모음
- 함수 만들기(정의하기) : 새 함수 이름 + 함수가 호출될 때 실행될 명령어들
  + 함수 안에 함수를 넣을 수 있음
  + 주의1 : 콜론(:)을 씀. 세미콜론 아님
  + 주의2 : 들여쓰기가 중요함
- 함수 실행하기(호출하기) : 함수 안에 있는 명령어를 수행
